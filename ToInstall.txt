## install SWARM
$ cd software
$ wget https://github.com/torognes/swarm/archive/refs/tags/v3.1.0.tar.gz
$ tar xzvf v3.1.0.tar.gz
$ cd swarm-3.1.0/
$ make
$ cp bin/swarm ../../bin/
$ swarm --version 
```
Swarm 3.1.0
```
done

## vsearch 
$ cd software
$ wget https://github.com/torognes/vsearch/archive/v2.21.1.tar.gz
$ tar xzf v2.21.1.tar.gz
$ cd vsearch-2.21.1
$ ./autogen.sh
##change PATH by your path (check using pwd)
$ ./configure CFLAGS="-O3" CXXFLAGS="-O3" --prefix=/PATH
$ make
$ make install
$ cp /bin/vsearch ../../bin/
$ vsearch --version
```
vsearch v2.21.1_linux_x86_64
```
done

##**SRA toolkit (can be useful for downloading SRA data)
$ cd software
$ wget http://ftp-trace.ncbi.nlm.nih.gov/sra/sdk/current/sratoolkit.current-centos_linux64.tar.gz
$ tar xzvf sratoolkit.current-centos_linux64.tar.gz
$ export PATH=$PATH/:$PWD/sratoolkit.2.11.2-centos_linux64/bin
$ fastq-dump --version

```
"fastq-dump" version 2.11.2
```
This command will download the SRA file SRR18042573 and save it the forward (.1) and reverse (.2) file in fastq.gz format. To Run from the Amplicon Pipeline folder
$ cd Amplicon_MiSeq_V4 
$ fastq-dump --split-files --gzip SRR18042573 --outdir $PWD/ExampleFiles/RawData/

##PR2 for taxonomic assignment of eukaryotes
$ cd Amplicon_MiSeq_V4
$ mkdir db_v4
$ cd db_v4
$ wget https://github.com/pr2database/pr2database/releases/download/v4.14.0/pr2_version_4.14.0_SSU_UTAX.fasta.gz
$ gzip -d pr2_version_5.0.0_SSU_UTAX.fasta.gz

##SILVA for taxonomic assignment of prokaryotes (PR2 can also be used but scripts need to be modified)
$ cd Amplicon_MiSeq_V4
$ mkdir db_v4
$ cd db_v4
$ wget https://www.arb-silva.de/fileadmin/silva_databases/current/Exports/SILVA_138.2_SSURef_NR99_tax_silva.fasta.gz
$ gzip -d SILVA_138.2_SSURef_NR99_tax_silva.fasta.gz
